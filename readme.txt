== Description ==
Simple Pin It Button is a WordPress plugin that adds a customizable "Pin it" button over images on hover with various options. It's designed to make it easy for your website visitors to share images on Pinterest, potentially increasing your site's social media engagement.

**Features**

* Customizable button color
* Multiple button sizes (small, medium, large)
* Choice of button shapes (rectangle, round)
* Flexible button positioning (top-left, top-right)
* Option to show on mobile devices
* Selective display on posts, pages, and archives
* Lightweight and easy to use
* Completely Free to Use All Features, No Need to Create any account.

**Third-Party Service**

This plugin uses Pinterest's service to create "Pin It" buttons. When a user clicks on a "Pin It" button, they will be directed to Pinterest's website to complete the pinning process.

- Pinterest website: https://www.pinterest.com
- Pinterest Terms of Service: https://policy.pinterest.com/en/terms-of-service
- Pinterest Privacy Policy: https://policy.pinterest.com/en/privacy-policy


== Installation ==
1. Upload the `Simple Pin It Button` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin settings under Settings > Simple Pin It Button

= Usage =
After activation and configuration, the Pin It button will automatically appear over images based on your settings. No additional action is required for basic functionality.
To customize the button's appearance or behavior:

* Go to your WordPress admin panel
* Navigate to Settings > Simple Pin It Button
* Adjust the settings according to your preferences
* Click 'Save Changes'

= Frequently Asked Questions =

Q: Can I control where the Pin It button appears?
A: Yes, you can choose to show the button on posts, pages, and/or archives through the plugin settings.

Q: Is the button mobile-friendly?
A: Yes, there's an option to show the button on mobile devices. You can enable or disable this in the settings.

Q: Can I change the style of the button?
A: Absolutely! You can customize the color, size, shape, and position of the button through the plugin settings.
